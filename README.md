Translator Python -> JavaScript

**Установка и запуск**
1. **Склонируйте репозиторий**:

git clone [<URL_репозитория>](https://github.com/KaraSikDarWin/Translator_Python_JS)  
cd <имя_папки_репозитория>

2. **Установите зависимости**:

bash
Копировать код
npm install

В этом приложении используются только стандартные зависимости React.

3. **Запустите приложение на локальном сервере**:

npm start

После этого приложение будет доступно по адресу http://localhost:3000.

4. **Сборка проекта**
Чтобы собрать приложение для продакшн:

npm run build
Собранные файлы будут находиться в папке build/.

Структура проекта
src/ — исходные файлы React-приложения

public/ — публичные файлы (HTML, favicon и т.д.)

5. **Требования**
Node.js >= 14
npm >= 6
